Excel AI Chatbot

This intelligent assistant is here to help you manage your Excel data.

Simply type your requests to view your data, add new rows, delete specific rows, or modify existing ones, and the chatbot will handle them for you.

When adding, please use the format {'column name': 'value', 'column name': 'value' ...}.

For modifying, include the row number followed by the new data using the format {'column name': 'value', 'column name': 'value' ...}.

If you wish to delete a row, just provide the row number.

Here are some examples:
Show me the data
I want to add a row {'A': '10', 'B': '20', 'C': '30'}
Please delete row 1
Modify row 2 to {'A': '77', 'B': '88', 'C': '99'}
